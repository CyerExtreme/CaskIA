# 🟠 CaskIA

> Asistente de inteligencia artificial local tipo Jarvis para Windows, construido con Python y modelos de lenguaje que corren 100% en tu máquina.

---

## 📋 Requisitos del Sistema

Para garantizar el correcto funcionamiento de la interfaz y la ejecución de los modelos, el equipo debe cumplir con las siguientes especificaciones:

* **Sistema Operativo:** Windows 10 o superior.
* **Memoria RAM:** Mínimo 8 GB de RAM (o superior).
* **Memoria de Gráfica (VRAM):** No es obligatoria para la configuración base, pero **sí es requerida** si se desea trabajar con modelos superiores o de mayor tamaño.

---

## ⚠️ IMPORTANTE — Aviso de Windows SmartScreen

Es posible que, al descargar y ejecutar CaskIA por primera vez, Windows te muestre una pantalla azul de advertencia llamada SmartScreen. Si esto ocurre, es completamente normal y no significa que el programa tenga virus.

**¿Por qué ocurre esto?**
Windows SmartScreen no analiza si un programa es peligroso o no. Simplemente comprueba si el ejecutable tiene una firma digital de pago de una empresa certificada. CaskIA es un proyecto personal sin ánimo de lucro y ese certificado cuesta dinero al año, algo que no puedo permitirme para una app que ofrezco gratis.

En pocas palabras: SmartScreen no sabe si el programa es malicioso, solo sabe que no ha pagado una firma digital. No es lo mismo. Si quieres comprobarlo tú mismo, el código fuente está disponible y puedes revisarlo.

---

## ✅ Cómo aceptar SmartScreen paso a paso

Cuando aparezca la pantalla azul:

1. Haz clic en **"Más información"** *(Está en letras pequeñas debajo del mensaje principal)*.
2. Haz clic en **"Ejecutar de todas formas"**.

Listo, CaskIA se abrirá normalmente.

---

## 🔁 Solo tendrás que hacer esto una vez

A partir de ahí CaskIA tiene un sistema de actualización automática integrado. Cuando haya una versión nueva te avisará dentro de la propia aplicación y se actualizará sola, sin que vuelva a saltar SmartScreen nunca más.

---

## 🛡️ ¿Puedo fiarme?

Sí. Estas son las razones:

* CaskIA está diseñada principalmente para correr de forma local en tu ordenador, aunque también permite la integración de modelos online si decides configurarlos.
* Los modelos locales de IA se descargan y ejecutan en tu propia máquina mediante Ollama.
* No hay telemetría ni recopilación de datos cuando utilizas los modelos locales.
* La aplicación se conecta a internet solo para buscar información cuando se lo pides, al usar modelos online, o para comprobar si hay actualizaciones disponibles.

---

## 💾 Instalación

1. Descarga el archivo `CaskIA.exe` desde la sección [Releases](https://github.com/CyerExtreme/CaskIA/releases).
2. Acepta SmartScreen como se explica arriba.
3. La primera vez, CaskIA instalará automáticamente todo lo necesario.

---

## 🔄 Actualizaciones

CaskIA includes un sistema de actualización automática. Cuando haya una versión nueva:

* Recibirás una notificación dentro de la app.
* Puedes aceptar o rechazar la actualización.
* Si aceptas, se descarga e instala sola sin que tengas que hacer nada.
* **SmartScreen no vuelve a saltar en ninguna actualización.**
